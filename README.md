# sentencepiece_proto

A simple C++ project with CMake for my learning purpose.

## Build Instructions

1. Create a `build` directory:
   ```bash
   mkdir build
   cd build
   ```

2. Run CMake to configure the project:
   ```bash
   cmake ..
   ```

3. Build the project:
   ```bash
   cmake --build .
   ```

4. Run the executable:
   ```bash
   ./sentencepiece_proto
   ```
